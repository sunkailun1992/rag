# 归档资料

本目录保存从 `utils` 公共 markdown 迁移过来的参考资料。AI 默认不需要阅读这里的文件，只有在追溯原始规范或错误码完整表时再打开。

## 文件说明

- `utils-markdown/Announcement.md`：团队环境、分支、乐观锁、数据库和枚举规则原文。
- `utils-markdown/ErrorCode.md`：完整错误码表。
- `utils-markdown/Specification.md`：压缩后的 Java 通用开发规范。

## 默认阅读入口

日常编码只读上层目录中的三份主规范：

1. `../AI_CODING_GUIDE.md`
2. `../PROJECT_CODING_SPEC.md`
3. `../UTILS_PUBLIC_SPEC.md`
